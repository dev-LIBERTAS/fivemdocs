# LIBERTASサーバー接続方法
1.FiveMを開きます。
![](https://github.com/user-attachments/assets/1ce508ce-62ff-415b-a337-9a3bed65acad)

2.開くことができたらF8キーを押して下記コマンドを貼り付けてEnterを押すことで接続できます。  

```
connect fivem.libertasmc.xyz
```
